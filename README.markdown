== README ==

How to instal:

1. Download the Mail.applescript file
2. Make a back-up of : "/Applications/iCal.app/Contents/Resources/Scripts/Mail.scpt
3. Open the Mail.applescript file in "AppleScript Editor"
4. Click compile
5. Copy the resulting Mail.scpt
